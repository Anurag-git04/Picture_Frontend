- Node.js (version 14+ recommended)  
- npm or yarn  

### Installation  
```bash
git clone https://github.com/Anurag-git04/Picture_Frontend.git  
cd Picture_Frontend  
npm install      # or yarn install  
npm run dev      # start the development server
```
